# Unemployment in India - Data Analysis

## About
Analysis of state-wise unemployment trends in India 
during the COVID-19 period (2020).

## Key Findings
1. National average unemployment rate: 11.79%
2. Highest unemployment region: Tripura
3. Lowest unemployment region: Meghalaya
4. Sharp unemployment spike during COVID-19 lockdown 2020
5. Urban regions showed higher unemployment than rural areas
